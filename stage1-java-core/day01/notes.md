# Day 1 - Java Introduction(JDK - JRE - JVM)
_Java là một ngôn ngữ lập trình hướng đối tượng (Object-Oriented Programming) được dùng để xây dựng:_
**Web backend**
**Mobile app Android**
**Hệ thống doanh nghiệp (enterprise systems)**
**Microservices / Cloud systems**
_Java được phát triển bởi Sun Microsystems vào năm 1995, sau đó được mua lại bởi Oracle Corporation_
